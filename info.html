Multi-player games (lottery draw, weekly raffle, rank winners, Hi-Lo card draws, Banana, VP lobby shared deck) are seeded by the bot's own PF stream — admins can verify via the bot's .verify.

Why so much variance? Bigger games make more rolls. A simple ,roll is one nonce because it's one die. A slot spin needs a roll per cell because each cell is independent — that's what makes it fair (and verifiable per-cell). The bot stores every nonce, so you can audit any roll at any time, no matter how many came before.
